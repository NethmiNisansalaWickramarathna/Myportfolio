# Professional Portfolio Website

Welcome to the source code of my personal portfolio website! I am an Information Technology (BIT) undergraduate at the University of Moratuwa, a passionate Unity game developer, and a full-stack enthusiast. 

This repository contains the front-end code that showcases my IT journey, software projects, and skills.

## ✨ Features
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile viewing using modern CSS.
* **Dynamic Project Filtering:** Easily sort through my Web Apps, Desktop Apps, and CLI Scripts using JavaScript DOM manipulation.
* **Integrated AI Chatbot:** A custom-built floating assistant that answers questions about my work, including my Unity games (like FoxRunner), my tech stack, and my document service, KN Docs.
* **Modern UI/UX:** Clean, professional aesthetics with a sticky navigation bar and interactive hover effects.

## 🛠️ Tech Stack
* **HTML5:** Semantic structuring of content.
* **CSS3:** Flexbox, CSS Grid, and Media Queries for responsiveness.
* **Vanilla JavaScript:** Custom logic for the AI chatbot and project filtering mechanisms.
